Data Binding


String Interpolation

  • String Interpolation inserts strings into the text within HTML elements
  • String Interpolation uses double curly braces and NO QUOTES
  • {{ interpolatedString }}
  • {{ interpolatedNumber | currency:'EUR':true }}
  • {{ interpolatedBoolean }}

Property Binding

  • Property binding sets an element's property to the value of a Template Expression
  • The binding target is enclosed in '[]'
  • The binding source is enclosed in ''

Event Binding

  • Event Binding listens for events and executes a template statement when the event occurs
  • The target for the event is enclosed in ()
  • The template statement is enclosed in ''
